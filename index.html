jQuery Line Length plugin

Example of the jQuery Line Length plugin. Plugin is developed for controlling text based on it's height and width parameters.

Plugin parameters include:

  • setHeight: Set the height that the plugin should attempt to make the element.
  • setWidth: Set the maximum width the element make expand to, if the element reaches the maximum width it will stop trying to alter the height..

CSS styling is example only.

Example one

Example on has a height set at 50px and a maximum width of 900px. It will reach the 50px height before the 900px width limit.

Example two

Example on has a height set at 100px and a maximum width of 500px. It will reach the 100px height before the 500px width limit.

Example three

Example on has a height set at 30px and a maximum width of 500px. It won't reach the 30px height before the 500px width limit.

Further information can be found at https://github.com/georgepaterson/jquery-linelength
